### Deskripsi
File ini merupakan contoh penggunaan modul multiprocessing di Python untuk melakukan pengumpulan tugas secara paralel. Mari kita jelaskan lebih detail dalam bentuk paragraf. Pada awalnya, kita mendefinisikan fungsi collectTask() yang mewakili tugas yang akan dikumpulkan oleh setiap proses. Di dalam fungsi ini, terdapat langkah-langkah yang akan dieksekusi oleh setiap proses, seperti mencetak pesan dan menunggu selama 3 detik sebagai simulasi pekerjaan yang dilakukan. Kemudian, di dalam blok if __name__ == '__main__':, kita membuat dua objek multiprocessing.Process yang akan menjalankan fungsi collectTask() secara paralel:process_with_name: Objek ini diberi nama "Pengumpulan Tugas 1" dan akan menjalankan fungsi collectTask().process_with_default_name: Objek ini tidak diberi nama khusus dan juga menjalankan fungsi collectTask().Selanjutnya, kedua proses tersebut diaktifkan menggunakan metode start(). Hal ini menyebabkan proses-proses tersebut mulai berjalan secara bersamaan.Kemudian, kita menggunakan metode join() pada kedua proses untuk memastikan bahwa program utama tidak berakhir sebelum kedua proses selesai dieksekusi. Dengan menggunakan join(), program utama akan terblokir hingga kedua proses selesai.Perlu dicatat bahwa pada kode tersebut, baris yang mengatur atribut daemon untuk process_with_name dijadikan komentar (#process_with_name.daemon = True). Jika Anda mengaktifkan baris tersebut, maka proses dengan nama "Pengumpulan Tugas 1" akan menjadi proses daemon. Proses daemon akan berhenti secara paksa jika program utama telah selesai, bahkan jika proses tersebut belum menyelesaikan tugasnya. Secara keseluruhan, kode tersebut memanfaatkan modul multiprocessing untuk menjalankan beberapa proses secara paralel, memungkinkan tugas-tugas untuk dikumpulkan secara efisien dengan memanfaatkan kemampuan pemrosesan paralel pada komputer yang tersedia.
